Mobile App Design and Automation of Hardware Assembly

Project Overview

This project demonstrates the integration of a 3D model, developed in SOLIDWORKS, into a mobile application. The web-based demo showcases how the model can be implemented and visualized within the app.

Live Demo

Experience the interactive 3D model by clicking the button below. Click on the model to open or close the lid, demonstrating the animation capabilities.

View 3D Model Demo

Technologies Used

  • SOLIDWORKS: For creating the detailed 3D model.
  • Three.js: JavaScript library used to render the 3D model in the browser.
  • GLTFLoader: A Three.js loader to load .glb (GL Transmission Format) files.
  • OrbitControls: Allows orbiting around the 3D model for better visualization.
